# Archived Backend Reference

`legacy-express-server.js` is the original Express prototype backend. It is kept here only as historical reference for the course project.

The active backend is the ASP.NET Core application in `Program.cs` and `Controllers/`, using PostgreSQL through Entity Framework Core. Do not run the archived Express server for current development.

Uploaded files and persisted local data were not deleted or moved as part of this archive step.
